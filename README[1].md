# AI-Based Clinical Decision Support System (CDSS)

## Project Overview
This project is a simple AI-based Clinical Decision Support System for healthcare.  
It predicts possible disease categories based on selected symptoms and gives basic recommendations.

## Objective
The objective of this project is to understand how Artificial Intelligence and Machine Learning can be used in healthcare to support decision-making, early screening, and treatment guidance.

## ML Algorithm Used
This project uses a Decision Tree-inspired approach.

A Decision Tree algorithm works by checking conditions step-by-step and classifying input data into an output class.  
For this project:
- Input: Symptoms selected by user
- Processing: Symptoms are compared with disease patterns
- Output: Possible disease prediction and basic guidance

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- User-friendly web interface
- Symptom selection
- Disease prediction output
- Basic health recommendation
- Educational medical disclaimer

## How to Use
1. Open the website.
2. Select the symptoms.
3. Click on "Predict Disease".
4. View the predicted condition and recommendation.
